# Design System Generator

I'm ready to help you build a complete design system for your project. Here's how we'll work together:

## Process Overview

I'll conduct a **structured brand interview** with you, asking questions one section at a time across these areas:

- **Brand Personality** — How your product should feel
- **Colors** — Primary, secondary, and accent choices
- **Typography** — Typeface and type scale
- **Spacing & Density** — Layout feel and whitespace approach
- **Reference Products** — Visual style benchmarks
- **Constraints** — Any limitations or requirements

Once you've answered all questions, I'll generate **two outputs**:

1. **design-system-visual.html** — A browser-viewable style guide showing all tokens (colors, type, spacing, shadows, radius) with usage labels
2. **/docs/design-system.md** — A structured specification file that Claude Code reads to apply your design system automatically

I'll also update your CLAUDE.md file to reference the new spec.
